# MarkView App Icon

**`markview-icon.svg`** is the single source of truth for the MarkView brand across all platforms.
Run `bash icons/generate.sh` to regenerate all platform icons and update the previews below.

## Choose M Padding

**Pick one below.** The M letterform uses SF Pro Bold (correct typographic proportions).
Smaller % = M fills more of the icon. Larger % = more breathing room.

### macOS / iOS (rounded rect)

| 26% padding | 28% padding | 30% padding | 32% padding |
|:-----------:|:-----------:|:-----------:|:-----------:|
| ![26%](previews/pad_26pct_macos.png) | ![28%](previews/pad_28pct_macos.png) | ![30%](previews/pad_30pct_macos.png) | ![32%](previews/pad_32pct_macos.png) |

### Android (circle mask)

| 26% padding | 28% padding | 30% padding | 32% padding |
|:-----------:|:-----------:|:-----------:|:-----------:|
| ![26%](previews/pad_26pct_android.png) | ![28%](previews/pad_28pct_android.png) | ![30%](previews/pad_30pct_android.png) | ![32%](previews/pad_32pct_android.png) |

> Reply with your choice (26/28/30/32) and it will be applied to all platforms.

## Regenerating Icons

```bash
# All platforms + README previews
bash icons/generate.sh

# After regenerating macOS, rebuild + install:
bash scripts/bundle.sh --install
```

> **Android:** adaptive icon uses XML Vector Drawables. Run `generate.sh --android` for scaled coordinates.
